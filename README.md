# Изменить размер фото

Простой веб-сайт для изменения размера изображений. Работает на телефоне и компьютере.

## Возможности

- Загрузка фото с устройства или перетаскиванием
- Изменение ширины и высоты
- Сохранение/отключение пропорций
- Быстрые кнопки: 25%, 50%, 75%, 800px, 1200px
- Сохранение результата

## Запуск

Открой `index.html` в браузере или запусти локальный сервер:

```bash
# Python 3
python3 -m http.server 8000

# Затем открой http://localhost:8000
```

## Технологии

- HTML/CSS/JavaScript (без фреймворков)
- Canvas API для обработки изображений
- Адаптивный дизайн для мобильных устройств
